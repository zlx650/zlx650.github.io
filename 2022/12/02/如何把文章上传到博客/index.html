如何把文章上传到博客

一段时间不写就老是忘记，记下来比较好。

1.打开安装目录

我的是（D:\myblog）

在文件夹中右键单击Git Bash Here，打开git命令窗口

![image-20221202175728804](C:/Users/Zheng ling x/AppData/Roaming/Typora/typora-user-images/image-20221202175728804.png)

2.新建一篇文章

在git命令窗口输入hexo new post “新建文章的名字”，回车

![image-20221202175907095](C:/Users/Zheng ling x/AppData/Roaming/Typora/typora-user-images/image-20221202175907095.png)

也可以直接写一篇markdown，在文章最前面插入，相关的title等设置好。

![image-20221202175945538](C:/Users/Zheng ling x/AppData/Roaming/Typora/typora-user-images/image-20221202175945538.png)

![image-20221202180044840](C:/Users/Zheng ling x/AppData/Roaming/Typora/typora-user-images/image-20221202180044840.png)

3.更新上传

在git命令行窗口输入

$ hexo clean    # 清空一下缓存，有时候博客页面显示不正常也可以试试这个命令行
$ hexo g    # 是hexo generate的简写，把刚刚做的改动生成更新一下
$ hexo server    # 在本地服务器看看博客有没有更新成NexT主题：https://localhost:4000
$ hexo d    # hexo deploy，如果本地服务器看了没问题就可以输入github账号和密码传到网站上啦
